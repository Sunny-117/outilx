# outilx

A collection of utility libraries for Node.js and Browser environments.

## Packages

- [@outilx/node](./packages/node) - Node.js utility functions
- [@outilx/browser](./packages/browser) - Browser utility functions

## Development

```bash
# Install dependencies
pnpm install

# Build all packages
pnpm build

# Run tests
pnpm test

# Release all packages
pnpm release
```
